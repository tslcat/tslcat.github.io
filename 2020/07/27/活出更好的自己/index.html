活出更好的自己

认识到自己很粘人是成功改变这种行为的第一步 。如果你很粘人，那么很可能你属于那种刚刚和某人相识就一头陷进去、迷恋对方的人。而对方可能是你的新恋人，也可能是你的新朋友。和喜欢的人在一起或者和喜欢的朋友结交之后，你总是喜欢频繁给对方打电话，叫对方出来和你玩。而如果对方想一个人呆着婉言谢绝，你就会觉得很沮丧，甚至有被抛弃的感觉。如果你有这些行为，或者你的朋友、恋人总是想让你多给他们一些空间，那么你就需要克服粘人的习惯，调整自己对待恋爱或朋友的态度。如果你有心改变，可以参考以下步骤。

把注意力放在自己身上

  1. 树立自信。许多人之所以太粘人，是因为他们对自己不满意，独处的时候总会缺乏安全感，有被抛弃感，觉得受到了忽视。这些人甚至会因为一些朋友出去玩没有叫他而纠结无比，因为他们会觉得没有人喜欢自己。所以一定要摒弃这些负面的想法，首先学会欣赏和喜欢自己。如果你充满自信，就不会轻易觉得别人要离开自己，也不会因为有人离开自己而痛苦不堪，这样也就不会那么粘人了。
    • 找到自己身上至少三个与众不同的优点。学会欣赏自己。
    • 享受你擅长的事物带来的乐趣。你可能擅长跑步，或者你在工作上非常能干，或者你很有幽默感，常常使人开怀大笑，这些都是优点。
    • 学会运用自信的人们的肢体语言。站立的时候身体要直，双臂不要放在胸前，并且尽可能地微笑。
    • 改正缺点。每个人都有缺点，而勇于面对自己的不足就是改正的第一步，也会让你自我感觉更好。
  2. 如果你不容易信任别人，就要在这方面做努力。许多人都有信任方面的问题，有的源自于童年被抛弃、忽视的经历，也有的可能源于最好朋友或者爱人的背叛。这些都会带来很大的痛苦，但是你要学会单独看待每一段关系，不要让过去影响你的现在。
    • 学会放下过去伤害过你的人和经历，将眼光放在眼前的这段关系，以及未来如何建立一段健康稳定的关系上。
    • 告诉自己，粘着对方并不能让对方更忠于你。事实上，粘着对方只会把你的另一半越推越远。
    • 不要纠结自己。你不可能一夜之间就解决自身的所有问题，但是只要意识到问题的存在，你就可以一点一点地付出努力，让自己的感受越来越好，更容易向他人打开心扉，并且不必以粘着对方为方法。
  3. 放松自己的焦虑情绪。很多粘人的行为追根到底其实是一种焦虑情绪的体现。你也许担心自己会一个人孤老终身，担心没有知心朋友，或者担心你一不在场，人们就会在背后嘲笑你。让你焦虑的也许仅仅是开始一段新的感情，所以你宁愿和不合适的人守在一起，担心结识新的朋友会带来种种困扰，所以就只守着那么几个认识的人，好让自己有安全感。
    • 很多时候，焦虑都和压力联系在一起。你也许感到焦虑，是因为你总是忙个不停，担心自己总有一天无法独自面对。因此，你可以尝试一些方法减轻压力，如冥想、瑜伽、保持健康的睡眠习惯等等，看看你的焦虑是否也会因此得到改善。
    • 如果你即将走进一个人很多的房间，那么进去之前先做几次深呼吸。告诉自己，和陌生人攀谈没什么大不了的，不要总是和几个熟人黏在一起。
  4. 主动找人谈谈。如果你发现自己过分依赖某人，不论是你的母亲，你的男朋友，还是你最好的朋友，那么你就需要找人谈一谈这个问题。你可以找一个关系比较近的朋友谈一谈，也可以和你的另一半，或者家人倾诉。如果你觉得自己的问题已经比较严重，还可以向心理咨询师寻求帮助，以免累积到一定程度会有引发抑郁症的危险。
    • 找人谈一谈，把憋在心里的话说出来，也可以帮助你找到问题的根源。表现得过于粘人可能有诸多因素，也许是从小家里兄弟姐妹众多，你总是得不到关注；也许是因为上一段恋情中你投入不够导致恋情失败等等。

学会管理人际关系

  1. 学会给别人空间。在任何一种人际关系中，给对方足够的空间都是维持一段关系健康发展的最好方法。不论是你的好朋友还是男朋友，都应该让对方有自己的时间喘口气。你应该明白，不总是黏在一起，才能让对方和你在一起的时候更加欣赏你。而如果你无时无刻都不想离开对方左右，那么迟早你们会对对方感到厌倦。毕竟，如果总是黏在一起，你们哪来的时间想念对方呢、回忆和对方在一起的点点滴滴呢？
    • 和别人交流的时候也是如此，要懂得给予对方空间，不要一发短信就没完没了，也不要不请自来地跑到人家那。这些做法都容易惹人生厌，甚至会让人觉得你不懂礼貌。要记住人际关系中互动是相互的，你来我往才是长久之计。
    • 别总缠着别人。如果你总是缠着别人，那么就会让人觉得你无处不在。关心一个人并不意味着时时刻刻要知道对方每天生活的一点一滴，这样人家就没有时间做自己的事了。
    • 有一个量化的方法，就是和一个人关系再好，各自独处的时间也要至少是两人共度的时间的三倍。如果你就是爱一个人爱得无可救药，那么也要明白，总是黏在一起，这种炽热的感觉不会持续多久的。
    • 独处的时候，正好可以做自己喜欢做的事，追求自己的兴趣。不要把独处的时间仅仅看做和你想要共度时光的人重聚之前需要打发的无聊时光。
    • 学会看懂别人的暗示。如果一个人需要自己带着，对方很可能不会那么积极地接你的电话，和你在一起的时候也可能有点不那么专心，或者告诉你他/她这周很忙等等。所以，如果发生此类情况，要懂得读懂对方的意思，不要粘着人家，而要尊重对方的意愿，给对方独处的空间。
  2. 如果结识了新朋友，那么要放慢节奏。许多比较粘人的人总会对新结识的人太过热情，不论是新结交的朋友还是仅仅约会过一两次的对象。这种做法说明你很害怕自己的好意得不到回报，觉得如果自己不加倍努力，对方就不会对你好。而事实却往往与此相反。所以还是放慢节奏，放松心情，尽量不要一开始就在一周内安排两次以上的会面。
    • 如果你发现自己的全部社交安排都是围着某人转，那么最终结果就很可能是把对方吓跑。
    • 不要刚和人家认识，就和对方掏心掏肺，倾诉你是多么需要朋友、男朋友、女朋友等等，这也会把人家吓跑。
    • 也别每次和新认识的朋友、对象出去都是你主动。双方互动的关系才是平等的交流。
  3. 不要像妈妈一样照顾任何人。许多粘人的人都觉得别人很需要他们照顾，所以总是围着对方转，喜欢给人家建议，即使人家没有问他。有时候他人的确需要你的帮助，但不要不分情况地对每个人嘘寒问暖，觉得没有你的帮助和建议别人就处理不好自己的事似的。
    • 如果有人需要你的帮助，他们会主动向你开口的，所以不要总觉得别人需要你。
  4. 注意你的身体语言。即使是不恰当的身体语言也会让你显得太粘人，仿佛你竭尽全力想要拉近和其他人的关系似的。如果你和朋友在一起，不要站得太近，也不要和对方有太多的肢体接触，或者随便玩弄人家的头发、饰品什么的，不然对方可能觉得你有点不知分寸。
    • 和另一半在一起的时候，偶尔相拥、亲吻都是很有爱的表现。但是你们也没有必要每分每秒都牵着手，出去玩或者参加聚会，也不要时刻黏在一起。
    • 和别人谈话的时候，给予对方足够的注意力是对的，但不要过分，不然死盯着人家看，或者不想让对方和其他人说话等都是不妥的行为。
  5. 不要让别人把你的付出当做理所应当。身为一个粘人的人，一大弊端就是，别人会把你的好当做理所当然的事，不懂珍惜。你的好朋友或者男女朋友知道，顶多稍加暗示，你就会马上跳出来帮助他们。如果你不想让人这样对待你，就不要让人觉得你总是有时间，时刻准备好要帮助他们。
    • 不要让你的整个社交生活都围着一个人转。在你容易粘着的那个人面前，让对方知道你的生活中还有别的朋友、有别的事情要做。
    • 可以提到你忙着做的其他事。是学校的研究项目也好，你参加的足球社的活动也好，或者仅仅是帮妈妈准备她的生日派对也好。只要让人知道，你很忙，日程排的满满的，不会只因为去见一个人而打乱你的计划。
    • 当然，你不应该随便放人鸽子。不过同时也要注意，不要别人打电话过来总是秒接，短信秒回，各种社交工具上的信息也是随时随地回复，不然就会显得你这个人好像没有别的事情做。
  6. 和他人保持正常、健康的距离，同时要学会享受这种距离给你带来的个人空间。一旦你学会了如何不去粘人，你就会渐渐喜欢上这种生活方式。这下子你有更多的时间用在自己的事情上，可以尽情追求你的爱好，朝自己的目标去努力。同时，当你和对方在一起共度时光的时候，双方也会更加享受和珍惜。
    • 多交朋友，同时和若干人建立友谊，享受和每个朋友在一起共度的时光，这样就不会把心思都花在一个人身上。
    • 隔一段时间就可以和朋友联系联系，不要担心，也可以问，“这周我没有把你缠得喘不过气吧”。一旦发现自己有粘着对方的趋势，就要立刻避免这样的行为。
    • 这样努力一段时间，你就会发现自己独处的时候是多么快乐，有那么多事情可以做，并不需要时时刻刻有人陪。学会独处才能更加吸引他人愿意和你靠近。

过更加充实的生活

  1. 追求自己的兴趣爱好。想要不再那么粘人，最简单的方法就是让自己忙起来，把生活变得精彩，充满乐趣，这样每天你都会迫不及待地去面对生活了。如果你总是无事可做，那么很自然地就总是想粘着你的另一半或者你的好朋友了。而如果你的生活充实有趣，总是充满了新鲜、有意义的活动，那么你就不那么容易变得粘人。以下就是一些切实可行的建议：
    • 找到你的兴趣所在。你肯定有至少一项爱好，不管是摄瑜伽，还是钢琴，也许你以前都没有意识到你是多么喜欢这项活动。不要害怕离开你的“舒适区”去尝试新鲜的活动，这样你的生活才会更加充实。
    • 找到一项有趣的运动。不论是慢跑、骑车登山，还是搏击操，都无不可。找到一项你感兴趣的运动可以帮助你释放多余的精力，同时让你自我感觉更好。如果你选择报班运动，那么一周一定要坚持去一到两次。这项有规律的新活动也会分散你的注意力，不会把心思都花在某个人身上了。
    • 每周都抽出几小时的时间用在你最喜欢的爱好上。不管你喜欢写歌还是写诗，喜欢园艺还是自制珠宝，最好每周都空出几个小时来做这些事。这样做不仅可以让你从喜欢的活动中获得成就感，还可以让你更加享受独处的时光。
  2. 专注于追求自己的目标。追求自己的目标和追求你的兴趣一样重要，都可以帮助你克服依赖他人的习惯。这样做可以帮助你把注意力放在眼前，专注于如何实现自己的短期梦想和长期的梦想上，而不是你想要依赖的人身上。不管你处于什么年纪，都应该有自己的短期目标和长期目标，来督促自己。
    • 建立短期目标。可以把目标定为简单的完成五千米跑，或者读完《战争与和平》等。你可以把目标日期写下来，让自己更加有动力。
    • 为你的长期目标制定计划。不论你的长期目标是以优异的成绩从大学毕业，是在工作中晋升，还是写一本书，都应该为之制定切实可行的计划，才能让你的梦想变成现实。这样一来你要考虑的事情自然就多起来了，也就不会光把注意力放在你的男朋友身上了。
    • 把你的目标以记日记的形式写下来。记日记可以帮助你更好地了解自己，也更明晰地看到美好的未来是什么样。这样你也就更加清楚地意识到，对自己来说，什么才是最重要的。
  3. 扩大自己的社交圈。这也是让你变得更加独立的方法之一。如果你只有两个朋友，或者每天见面的只有你的另一半，那么你就自然更容易会粘人了。你的社交圈越广，认识的人越多，你的社交经验也就更丰富，就更不容易把全部心思都花在某个人身上了。以下就是一些扩大社交圈的建议：
    • 所谓扩大圈子，不是说你必须有十个无话不谈亲密无间的好朋友才行。你需要做的只是和关系一般的朋友偶尔喝个咖啡，拉进一下距离，或者将仅仅是认识的人变成普通朋友，这样一来你的圈子自然也就扩大了。
    • 对待同事或同学要友好。这样普通的共事关系也可以变成一段友谊。即使只是每两个月和同事小聚一下，你的社交圈也就不知不觉地扩大了。
    • 和老朋友叙叙旧。也许你忽略了你的老朋友，仅仅因为你把注意力都放在了某个人身上。重新和他们联络起来吧，也可以为你的失踪二道歉。
    • 不要害怕约朋友出去。如果你在一次聚会中和另一个女生谈得来，就可以问问她愿意不愿意和你一起去上瑜伽课，或者一起喝一杯之类的。
  4. 学会享受独处。许多不够独立的人99%的时间都想和别人呆在一起。而总是想找人陪，你就更没有机会独处了。一定要学会享受独处的时间，这样对你建立自信非常重要。仔细想想什么事会让你感到开心，然后就找机会一个人做这件事。以下就是几条关于独处的建议：
    • 独自漫步。这项活动不仅利于健康，还可以帮助你更好地思考。
    • 学会享受阅读的乐趣。阅读不仅愉悦身心，可以让你学到更多的东西，而且一本书本身就是很好的伙伴。
    • 重新装饰你的住所。重新装饰你的住处不仅可以让心情更好，也会让你更加享受独处的时光。
  5. 参加志愿者活动。志愿者活动让你更好地帮助他人，还可以让你获得满足感和被需要感，这样也就不会总觉得你需要别人了。可以志愿打扫公园、在社区图书馆教课等等。
    • 当你找到了喜欢的活动，每周至少要去一到两次。这种方法会让你忙起来，你的日程排的满满的，自然也就没有工夫去依赖他人了。
    • 志愿者活动还可以帮助你扩大你的社交圈。你也许会发现有非常合得来的人，这样也就交到新朋友了。

部分小提示

  • 分开独处其实可以给两颗心更多的空间彼此靠近。你生活中的每个人对你来说都很重要，你对他们也是如此。你们给予彼此适当的空间越多，其实更能拉近感情，因为不是无时无刻不在一起，在一起时也就更懂得珍惜。这样做别人也就更能理解你对个人空间的需要，而人们彼此有了适当的空间，人际关系才能健康发展。
  • 要及时发现自己不独立的粘人行为。你可能会从你依赖的对象身上发现不耐烦和困扰的迹象。不要试图利用对方的情绪进行进一步的依赖，而要自我反省自己的行为，找到更好的与他人沟通的方法，这样做不仅给了对方空间，也给了自己解决问题的空间。
  • 如果你的朋友或者另一半是个比较粘人的人，那么就可以建议一些分开进行，同时又是对方感兴趣的活动。这样可以逐渐让对方认识到，除了和他人在一起，还有更多的有趣的事是可以自己独自进行的。从一起参加的活动过渡到独处的活动也需要一个过程，因此一定要耐心帮助对方完成这个过渡。
  • 如果你连最基本的个人生活都需要依赖他人，那就要认真地问问自己为什么了。你可能有一定程度的恐惧症或者社交障碍等等，需要寻求心理医生的帮助和治疗。同样，如果是真的出于生理原因不得不依赖他人，也不要觉得你是别人的负担。尽可能地从多渠道获得帮助，尤其是专业帮助，这样也会更加有效，可以帮助你完成独自无法完成的事。

部分警告

  • 长期地依赖他人会让别人的耐性慢慢磨光，即使再亲近的人也是如此。假以时日，即使是最有耐心的人也会受不了这种关系的，因此一定要及时发现自己的依赖倾向。
