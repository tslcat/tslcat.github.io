ubuntu下搭建ftp服务器实现上传于下载功能

Ubuntu自带的FTP服务器是vsftpd. 该服务器比较小，适合直接安装

  • 安装vsftpd
    确保已连接网络的情况下，在Terminal中执行sudo apt-get install vsftpd
    安装了之后会在/home/下建立一个ftp目录。可以在该目录中新建一个文件用于后续的测试。
  • 访问新建的ftp服务器
    可以先看一下机器的ip地址，在Terminal中执行ifconfig，查看机器的ip地址。
    然后可以在浏览器的地址栏输入ftp://IP地址。
    如果是在本机访问的话，可以在浏览器的地址栏输入ftp://127.0.0.1:21。
    输入用户名和口令（这里和登陆Ubuntu的那个用户名和口令相同），就应该能够看到刚才新建的那个文件了。新建的ftp服务器是支持下载的，因此可以把看到的文件下载下来。
  • 如何开启ftp服务器的文件上传功能
    新建的ftp服务器是不支持上传功能的，如果你使用的是ftp命令行方式或ftp客户端方式连接服务器，并要上传文件的话，就需要开启用户上传文件功能。步骤如下:
    打开vsftpd的配置文件/etc/vsftpd.conf
    将chroot_local_user=YES和write_enable=YES前面的“#”去掉。
