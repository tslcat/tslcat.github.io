十步流程——微软live、msn等后缀邮箱注册方法

准备条件

  • 需要一个outlook.com或hotmail.com后缀的微软账号

  • 火狐浏览器 firefox.com.cn

  • Tamper Data :: Firefox 附加组件

流程

  1. 火狐浏览器上登录你的微软邮箱帐号：
    https://account.microsoft.com/?lang=zh-CN

  1. 添加别名，转到网址：https://account.live.com/AddAssocId

  2. 火狐工具里打开Tamper Data，并点击 Start Tamper

  1. 先尝试注册自己想要的名字，确保可用，如果提示已被占用则后续怎么做都是无用功。

  2. 添加别名选择第一个，点击添加别名，

  3. 此时，Tamper Data 弹出篡改窗口，当然要点确认篡改

  4. 在弹出窗口，SingDomain那一项输入live.com或cn或msn.com

  5. 然后点submit确认，连续点击几次，直到不再弹窗，页面后缀也变化成需要的

  6. 然后继续点击添加别名，弹窗之后继续点击submit

  7. 持续点击submit，直到添加成功
